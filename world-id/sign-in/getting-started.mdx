## Sign In with World ID Template


<CardGroup>
<Card title="Use this Template" icon="link" href="https://github.com/new?template_name=world-id-nextauth-template&template_owner=worldcoin" horizontal />
</CardGroup>

Once you've created a new repository from this template, set your `NEXTAUTH_URL`, `WLD_CLIENT_ID` and `WLD_CLIENT_SECRET` in the `.env` file and run the following command to get started:

```yaml Usage
pnpm i && pnpm dev
```

To understand deeper our OIDC implementation check out [explainer page](https://docs.world.org/world-id/sign-in-with-world-id/oidc).
